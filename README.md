# Bank-Loan-Risk-Performance-Analysis
This project focuses on analyzing bank loan data to evaluate loan performance, identify default risk patterns, and derive actionable business insights. The analysis helps financial institutions understand borrower behavior, assess credit risk, and improve lending decisions.

# Business Problem
Banks issue thousands of loans every year, but loan defaults can significantly impact profitability.
The objective of this project is to:

-Analyze loan application and repayment data
-Identify key factors contributing to loan defaults
-Compare good vs bad loans
-Evaluate loan performance across borrower segments
-Support data-driven risk assessment and decision-making.

# Tools & Technologies
-Python
-Pandas – data manipulation
-NumPy – numerical operations
-Matplotlib & Seaborn – data visualization
-Plotly – interactive visualizations
-Power BI - dashboard creation
